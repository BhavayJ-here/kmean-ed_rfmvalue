# kmean-ed_rfmvalue
Customer segmentation using RFM features, KMeans clustering, FastAPI, and SQLite

this project takes recency, frequency and monetary out of uncleaned dataset and does behaviour analyses of customers

this makes them being clustered by how recently they bought, how often, and how much they spent.

the dataset was of online retail by UC irvine

Stack: sklearn, matplotlib, Kmeans, FastApi, SQLite, Bootstrap

run:
main.py
then uvicorn api:app --reload
and then localhost for dashboard
